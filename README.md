# TensorFlow-in-Action

  Buku "TensorFlow in Action" oleh Thushan Ganegedara adalah panduan praktis untuk mempelajari TensorFlow versi 2.9 dalam penerapan machine learning dan deep learning. Buku ini ditujukan bagi praktisi ML dengan berbagai tingkat pengalaman, mulai dari pemula hingga menengah, yang ingin menguasai TensorFlow untuk membangun solusi ML yang efektif dan produksi-siap.

Tujuan buku ini adalah memberikan pemahaman mendalam tentang ekosistem TensorFlow, dari konsep dasar hingga implementasi model canggih seperti CNN, RNN, dan Transformer. Selain itu, buku ini menekankan praktik terbaik dalam membangun pipeline data, memonitor model, dan mendeploy solusi ML dengan TFX.

# Struktur Buku (Per Bab)  
Part 1: Foundations of TensorFlow 2 and Deep Learning
1. The Amazing World of TensorFlow  
   Pengenalan TensorFlow, perbandingan CPU/GPU/TPU, kapan menggunakan dan tidak menggunakan TensorFlow, serta tujuan buku.

2. TensorFlow 2  
Dasar-dasar TensorFlow 2: eksekusi eager, AutoGraph, tf.Variable, tf.Tensor, tf.Operation, dan operasi umum dalam neural network.

3. Keras and Data Retrieval in TensorFlow 2  
Tiga API Keras (Sequential, Functional, Sub-classing) dan cara mengambil data menggunakan tf.data, Keras DataGenerators, dan tensorflow-datasets.

4. Dipping Toes in Deep Learning  
Implementasi dan pelatihan Fully Connected Networks (Autoencoder), Convolutional Neural Networks (CNN), dan Recurrent Neural Networks (RNN) untuk tugas nyata.

5. State-of-the-Art in Deep Learning: Transformers  
Pengenalan model Transformer, representasi teks, dan komponen utama seperti self-attention dan multi-head attention.

Part 2: Look Ma, No Hands! Deep Networks in the Real World

6. Teaching Machines to See: Image Classification with CNNs  
Analisis data gambar, pipeline data dengan ImageDataGenerator, implementasi Inception Net, dan evaluasi model.

7. Teaching Machines to See Better  
Teknik reduksi overfitting (augmentasi, dropout, early stopping), implementasi Minception, transfer learning, dan interpretasi model dengan Grad-CAM.

8. Telling Things Apart: Image Segmentation  
Segmentasi gambar dengan DeepLabv3, atrous convolution, pipeline tf.data, fungsi loss, metrik evaluasi, dan pelatihan model.

9. Natural Language Processing with TensorFlow: Sentiment Analysis  
Pemrosesan teks, pipeline NLP, analisis sentimen dengan LSTM, dan penggunaan word embeddings.

10. Natural Language Processing with TensorFlow: Language Modeling  
Pemodelan bahasa dengan GRU, generasi teks, evaluasi kualitas teks, dan beam search.

Part 3: Advanced Deep Networks for Complex Problems  

11. Sequence-to-Sequence Learning: Part 1  
Penerapan model seq2seq untuk machine translation (Inggris-Jerman) dengan encoder-decoder.

# Target Pembaca
* Praktisi ML (data scientist, ML engineer, researcher) dengan pengalaman dasar Python dan ML.
* Mahasiswa atau profesional yang ingin mendalami TensorFlow untuk tugas klasifikasi gambar, NLP, segmentasi, dan deployment.
* Pembaca yang ingin memahami best practices dalam membangun pipeline ML yang scalable dan maintainable.

# Kelebihan Buku
* Praktis dan aplikatif dengan contoh kode yang siap dijalankan.
* Cakupan luas dari dasar hingga topik lanjutan (Transformer, TFX, MLOps).
* Panduan terstruktur dengan penekanan pada kode produksi-siap.
* Akses ke kode sumber dan forum diskusi online.

Buku ini cocok sebagai referensi utama bagi siapa saja yang ingin menguasai TensorFlow secara mendalam dan menerapkannya dalam proyek ML nyata.
